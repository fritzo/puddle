## Puddle-hub

HTML5 app to be served by puddle-server


###Features:    

    [ ] Display corpus
    [ ] Edit corpus
    [ ] Uses cutting edge packages and interfaces
    [ ] HTML5 editor
    [ ] Uses puddle-editor-wrapper as interface
    [ ] Syncs in realtime
    [ ] Works in latest Chrome, FF
    [ ] no Opera or IE
    [ ] no previous versions tested
