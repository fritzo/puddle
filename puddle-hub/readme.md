## Puddle-hub

Socket.io adapter for puddle-crud to sync data server<=>browser


###Features:
    [X] Minimum dependencies
    [X] Server <=> Client sync               
    [X] Conforms to Corpus CRUD API    
    
    
###Installation:
    
    npm install puddle-hub
    npm test        # optional
    
###Usage:    
    
    var hub = require(‘puddle-hub’);
